## Project Overview

Web downloader tool. Python 3.10+ project managed with [uv](https://docs.astral.sh/uv/).

## Commands

```bash
# Run the main entry point
uv run python main.py

# Add a dependency
uv add <package>

# Add a dev dependency
uv add --dev <package>
```
